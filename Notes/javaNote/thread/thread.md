# Process
        ```aidl
            Java用Thread对象表示一个线程，通过调用start()启动一个新线程；
                一个线程对象只能调用一次start()方法；
                线程的执行代码写在run()方法中；
                线程调度由操作系统决定，程序本身无法决定调度顺序；
                Thread.sleep()可以把当前线程暂停一段时间
        ```
